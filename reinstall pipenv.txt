# щоб перевстановити pipenv перелік кроків
deactivate          виходимо з віртуального оточення
pipenv --rm         видаляємо віртуальне оточення
mkdir .venv         наново створюємо
Pipfile.lock        теж видаляємо

pipenv shell        запускаємо pipenv
pipenv install      інсталюємо віртуальне оточення
